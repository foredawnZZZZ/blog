## 概述
运算符是处理数据的基本方法，用来从现有的值得到新的值。JavaScript 提供了多种运算符，覆盖了所有主要的运算。

- 算术运算符
- 比较运算符
- 布尔运算符
- 二进制位运算符
- 其他运算符和运算顺序

## 算术运算符
A:
```javascript
/*
递增(递减)操作符:首先都对原始值进行修改
  - 前置递增(递减) 先修改原始值在执行语句
  - 后置递增(递减) 先进行执行语句在修改原始值
  - 除了整数还是适用浮点数,布尔值,字符串,对象
*/
var x = 12;
var y = 3;
console.log(x++ + --y);
console.log(x++ + ++y);
console.log(x-- - y--);
console.log(x + y--);
console.log(x);
console.log(x-- - --y + ++x);
console.log(x + ++y);
console.log(y);

---------------------------------------------
/* 
具体规则:
  - 当包含有效数字字符的字符串,先转为数值在进行语句执行 执行后变成数值!
  - 当包含非有效数字字符的字符串,直接给NaN! 执行后也是NaN!
  - 当为布尔值是 按照true为1 fasle为0 进行语句执行 执行后变成数值!
  - 浮点数跟整数执行一样
  - 当为对象时 先调用valueOf方法 取得的值是数值在执行 如果是NaN在调用toString 在执行最后的值都是数值
*/
var a = "12";
var b = '1';
var c = "foredawn"
var flag1 = true;
var flag2 = false;
var obj = {
  valueOf: function () {
    return "1"
  },
  toString: function () {
    return 1
  }
};
console.log(flag1++ - 2);
console.log(flag1);
console.log(--flag2 + 1);
console.log(flag2);
console.log(a++ + b--);
console.log(a);
console.log(b);
console.log(c++);
console.log(c);
console.log(++obj);

---------------------------------------------
/* 
  数值运算符:
  - 对于非数值类型运算 根据Number()转型规则解析 因为他是根据Number()来的 所以执行后变成数值了!!
*/
var a = 12;
var b = "13";
var c = 1.1;
var flag = true;
var arr = [];
var obj = {};
console.log(+a);
console.log(+b);
console.log(+c);
console.log(+flag);
console.log(+arr);
console.log(+obj);
console.log(-a);
console.log(-b);
console.log(-c);
console.log(-flag);
console.log(-arr);
var obj1 = {
  valueOf:function() {
    return 3;
  }
}
console.log(-obj);
console.log(-obj1);
```

B:
```javascript
/*
乘性操作符: 乘 除 求模
  - 乘 * :计算两个数值!之间的乘积 在遇到非数值时的运算规则
    1. 乘积超过了数值取值范围 返回+-Infinity
    2. 如果一个是NaN 返回NaN
    3. Infinity与0相乘是NaN
    4. Infinity与非0相乘 返回+-Infinity
    5. 如果有一个操作数不是数值 就根据Number()进行转换 然后在根据上规则
  - 除 / : 规则跟上面的差不多
    1. 如果Infinity跟Infinity相除 返回NaN
    2. 0/0与0%0返回NaN
  - 求模 %: 敲了挺长时间代码了没用过...先不写了
*/
var a = 10;
var b = 2;
var c = "12";
var d = "foredawn";
console.log(a * b);
console.log(a * c);
console.log(a * d);
console.log(a * Infinity);
console.log(0 * Infinity);
console.log(0 % 0);
console.log(0 / 0);
console.log(Infinity % Infinity);
console.log(Infinity * Infinity);

---------------------------------------------

/*
加性操作符: 加 减
  - 加 + :正常加就行 遇到非数值时规则
    1. 一个操作数是NaN 就返回NaN
    2. Infinity+Infinity(-Infinity+-Infinity) 返回+-Infinity
    3. 如果一个操作数是字符串? 首先只有一个字符串就把另一个转为字符串然后拼接 其次两个都是字符串就直接转
    4. 如果操作数有布尔,对象就调用toString方法 根据规则拼接 如果遇到null和undefined就调用String转型(都是引擎自后台整哒)
  - 减 - : 差不多 记住下面的就行 主要是+有拼接
    1. 如果一个操作数是字符串,布尔,null,undefined 先根据Number()转型 在进行减法 注意NaN
    2. 如果是对象 先调用valueOf或者toString 返回结果进行减法 
*/
var x = 10;
var y = 15;
var z = "12";
var n = "foredawn";
console.log(x + y);
console.log(x + z);
console.log(x + n);
console.log(Infinity + Infinity);
console.log(-Infinity + -Infinity);
console.log(true + z);
console.log(null + z);
console.log(undefined + z);
console.log(n - z);
console.log(y - z);
console.log(null - y);
console.log(undefined - y);

/*
赋值操作符: = += -= /= *=
 - 简单的赋值操作就是 = 把右边的值给左侧
 - 复合赋值操作就是 下面
*/
var num = 10;
num += 10;//num = num + 10
console.log(num)
